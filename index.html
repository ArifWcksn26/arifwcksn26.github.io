About Me

Perkenalkan, saya Arif Wicaksono — seorang Fresh Graduate S1 Sistem Informasi dari Universitas Pamulang dengan minat kuat pada Web Development dan pengolahan data. Selama studi, saya aktif membangun berbagai aplikasi berbasis web dan mendalami bagaimana teknologi dapat memberikan solusi yang efektif bagi kebutuhan pengguna.

Saya memiliki pengalaman dalam pengembangan aplikasi menggunakan Laravel dan Filament, seperti sistem pembayaran SPP digital dan sistem informasi akademik untuk pengelolaan nilai siswa. Salah satu proyek yang paling saya banggakan adalah NginapAja, sebuah platform pencarian dan pemesanan tempat tinggal yang saya bangun secara full stack, mulai dari perancangan antarmuka, implementasi backend dengan arsitektur repository Laravel, pengelolaan database, hingga deployment aplikasi. Proyek ini memberi saya pengalaman menyeluruh dalam pengembangan sistem berbasis web.

Selain Web Development, saya juga mendalami pengolahan data dan analisis data menggunakan Python dan R. Saya berpengalaman melakukan data cleaning, exploratory data analysis (EDA), serta membuat visualisasi untuk mengidentifikasi pola dan insight. Pengalaman ini membantu saya memahami proses pengambilan keputusan berbasis data secara lebih menyeluruh.

Saya memiliki Sertifikasi Pemrogram BNSP diterbitkan pada Juli 2025 dan berlaku hingga Juli 2028, sebagai bukti kompetensi profesional di bidang teknologi informasi, serta aktif mengikuti berbagai pelatihan untuk meningkatkan keterampilan, termasuk di Dicoding Indonesia, DQLab, dan RevoU dalam bidang backend development, web development, dan pengolahan data. Dengan pengalaman proyek yang solid serta pemahaman menyeluruh terhadap sistem, saya siap berkontribusi dalam pengembangan aplikasi, pengelolaan data, maupun solusi teknologi lainnya.
