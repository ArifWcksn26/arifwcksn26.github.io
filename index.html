About Me

Halo, saya Arif Wicaksono. Saya adalah Fresh Graduate S1 Sistem Informasi dari Universitas Pamulang dengan fokus pada Web Development, saya sangat termotivasi untuk terus belajar dan berkontribusi di industri teknologi yang dinamis.

Saya mengembangkan aplikasi berbasis Laravel dan Filament, termasuk sistem pembayaran SPP digital serta sistem informasi akademik untuk pengelolaan nilai siswa. Saya juga mengembangkan NginapAja, platform booking tempat tinggal berbasis arsitektur repository Laravel. Selain pengembangan web, saya tertarik pada bidang Data Science, menggunakan Python dan R untuk analisis serta visualisasi data, termasuk membuat dashboard interaktif menggunakan Streamlit untuk eksplorasi data secara menyeluruh.

Saya berkomitmen untuk terus belajar dan mengeksplorasi berbagai aspek teknologi yang menarik, sebagai seorang pemula yang berusaha ingin menjadi ahli dalam bidangnya, semoga diberi kesempatan untuk berkontribusi dan belajar dari pengalaman nyata di industri.
